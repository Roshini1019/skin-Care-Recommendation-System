# Skin Care Recommendation System

## Overview

The Skin Care Recommendation System is a web-based application that suggests suitable skincare ingredients and product types based on a user's skin type and skin concerns. The goal of the system is to help users make better skincare choices by recommending appropriate ingredients and avoiding harmful ones.

## Features

* Skin type selection (Oily, Dry, Combination, Normal, Sensitive)
* Multiple skin concern selection
* Recommended skincare ingredients
* Ingredients to avoid
* Cleanser (Face Wash) recommendation
* Moisturizer recommendation
* Sunscreen recommendation
* Reset option to perform a new analysis
* Simple and user-friendly interface

## How the System Works

1. The user selects their skin type.
2. The user chooses one or more skin concerns.
3. The frontend sends this information to the Flask backend.
4. The backend processes the data and determines suitable ingredients.
5. The system displays recommended ingredients, ingredients to avoid, and suitable skincare product types.
